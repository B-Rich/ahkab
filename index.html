Ahkab

This project aims to develop a SPICE-like electronic circuit simulator. The code should be easy to read and modify, the main language is Python and it is platform-independent.

News

  • (21/08) Hi GitHub! The project moved from the old storage at Google Code to GH.
  • ahkab can now be run stand-alone with a netlist file or within Python scripts as a library. This will likely become the preferred way in the future, but there are no plans to discontinue the command line interface.
